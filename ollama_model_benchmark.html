Benchmark Information

Test Categories

Coding & Fundamentals (12 tests) - Basic and intermediate coding skills: algorithms, data structures, language features, SQL, and framework knowledge.

Agentic Reasoning (4 tests) - Advanced problem-solving: multi-step planning, error handling with constraints, system design thinking, and analysis before implementation.

How Scoring Works

  • PASS - All validation checks passed. Code demonstrates proper understanding.
  • PARTIAL - Some checks passed. Code shows partial understanding or incomplete implementation.
  • FAIL - No checks passed. Response does not demonstrate required concepts.

Each test contains 3-4 regex-based checks that validate key aspects of the solution (function names, algorithms, language features, etc.).

Benchmark Settings

  • Temperature: 0.1 (low randomness for consistent results)
  • Max Token Output: 500 tokens per response
  • API Endpoint: http://localhost:11434

These settings are optimized for consistent testing of core problem-solving ability.

Metrics Explained

  • Success Rate: Percentage of tests that received full PASS score
  • Response Time: Wall-clock time from request to complete response
  • Tokens/sec: Model throughput (tokens generated per second)
